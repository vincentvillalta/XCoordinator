AnyRouter

public final class AnyRouter<RouteType> : Router where RouteType : Route

AnyRouter can be used as an abstraction from a specific Router implementation without losing type information about its RouteType.

This type abstraction can be especially helpful when injecting routers into viewModels. AnyRouter abstracts away any implementation specifics and reduces coordinators to the capabilities of triggering routes.
